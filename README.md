# StudyFlow - Protótipo de Aplicativo

Protótipo desenvolvido para a disciplina **Interação Humano-Computador (IHC)** do curso Técnico em Desenvolvimento de Sistemas – IFSULDEMINAS.

## Descrição

O StudyFlow é um protótipo de aplicativo mobile para organização de tarefas acadêmicas.  
O objetivo é demonstrar o fluxo de interação do usuário e a navegação entre telas, antes da implementação completa de um sistema real.

## Funcionalidades simuladas

- Tela de login
- Visualização de lista de tarefas
- Criação de nova tarefa
- Navegação entre telas
- Feedback visual ao adicionar tarefas

## Estrutura do protótipo

O protótipo possui três telas principais:

1. Login do usuário  
2. Lista de tarefas  
3. Criação de nova tarefa  

Fluxo de navegação:

Login → Lista de tarefas → Nova tarefa → Lista atualizada

## Objetivo

Demonstrar conceitos básicos de prototipação de interface e usabilidade, aplicando princípios estudados na disciplina de Interação Humano-Computador.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
